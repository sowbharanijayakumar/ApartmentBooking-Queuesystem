Apartment Booking Queue Management System

## About
Java Swing based Apartment Booking System 
using DSA concepts.

## Tech Stack
- Java
- Java Swing (GUI)
- Data Structures (PriorityQueue, Queue)
- OOP Concepts

## Features
- Priority Booking (Senior Citizen First)
- FIFO Queue Management  
- Real-time Admin Panel
- Token Generation System

## How to Run
javac Customer.java ApartmentQueue.java 
      AdminPanel.java MainDashboard.java
java MainDashboard

## DSA Concepts Used
- PriorityQueue
- Queue (FIFO)
- ArrayList
- Comparable Interface
